  • If you run this in a large environment it takes a while for some of the queries to come back. I haven’t had a chance to add a progress bar yet so be patient please. Especially if you click “Get Current App Devices” it has to do an API call for each device the app is installed on so it can take a while. Eg. in a large environment where they have acrobat deployed to 3000 devices it takes about 5 minutes. This delay is for the first time running reports since the device list is cached in the app. So, the next time you run the report it will only do API calls for devices which are new.
