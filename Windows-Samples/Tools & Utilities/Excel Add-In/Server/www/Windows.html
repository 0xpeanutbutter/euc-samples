Workspace ONE Windows Excel Add-In (Beta)

Installation

This is the installation documentation page for the Workspace ONE Windows Excel Add-In.


  • Contact Leon Letto for the add-in xml file.

  • See the following web page from Microsoft which shows step by step how to install on Excel for Windows

    Sideload Office Add-ins for testing

  • After this you need to add the following registry key:

    [HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Office\16.0\Wef]

    Then add the following dword value in the above key.

    "AlertInterval"=dword:00000000

    You need to do this when sideloading an app to tell Excel not to timeout on long queries.
